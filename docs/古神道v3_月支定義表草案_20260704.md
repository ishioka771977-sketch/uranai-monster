## 定義表(各支2行: ①理論根拠 ②顕れ方の体感語)

| 支 | ①理論根拠(五行/季節/蔵干) | ②顕れ方(体感語・生成プロンプトに渡す行) |
|---|---|---|
| **子** | 水・仲冬(大雪〜)。本気癸(雨露)・余気壬。一陽来復の月 | 誰も見ていない夜のうちに、最初の一滴が動き出す。**静かな始動**——顕れは水面下から |
| **丑** | 土・晩冬(小寒〜)。本気己(田畑)・中気辛・余気癸。凍土が春を待つ | 凍った土の下の貯蔵庫。**溜めてから動く**——顕れは満ちるまで出てこない |
| **寅** | 木・初春(立春〜)。本気甲(大樹)・中気丙・余気戊。火種を抱いた発芽 | 夜明けと同時に踏み出す。**火種ごと進む**——顕れは最初の一歩の勢いに出る |
| **卯** | 木・仲春(啓蟄〜)。本気乙(草花)・余気甲。草木全開の月 | 硬い幹ではなく、枝葉として四方へ。**しなやかに広がる**——顕れは柔らかさと数 |
| **辰** | 土・晩春(清明〜)。本気戊(山岳)・中気癸・余気乙。水と木を抱く湿土=龍の器 | 異質なものを腹に呑んで、うねりにする。**混ざりを束ねる**——顕れは編集力 |
| **巳** | 火・初夏(立夏〜)。本気丙(太陽)・中気庚・余気戊。点火と上昇 | 表に出る前に、内側で先に燃えている。**内燃から外へ**——顕れは温度差の一撃 |
| **午** | 火・仲夏(芒種〜)。本気丁・中気己・余気丙。正午の火勢 | 真昼の道に影はない。**直進して照らす**——顕れは隠さないこと・即答 |
| **未** | 土・晩夏(小暑〜)。本気己・中気乙・余気丁。余熱を含む乾土 | 火を止めたあと、味がしみていく時間。**熟成させる**——顕れは引き際のあとに残る |
| **申** | 金・初秋(立秋〜)。本気庚(剛鉄)・中気壬・余気戊。実が硬化する月 | 使う道具を素早く持ち替える。**切り替えの速さ**——顕れは手数と実務 |
| **酉** | 金・仲秋(白露〜)。本気辛(宝石)・余気庚。収穫と研磨 | 九割できたものを、最後の一磨きで別物にする。**仕上げの純度**——顕れは削ぎ落とし |
| **戌** | 土・晩秋(寒露〜)。本気戊・中気丁・余気辛。火を蔵して冬に備える燥土 | 消えかけの火種を懐で守る番人。**締めて守る**——顕れは戸締まりと継続 |
| **亥** | 水・初冬(立冬〜)。本気壬(大海)・中気甲・余気戊。水中に種木を抱く淵 | 深く潜って、次の春の仕込みをする。**潜って仕込む**——顕れは見えない準備 |

## 設計メモ

1. **十干(守護神)との役割分担**: 十干=「何を意識するか」(意識の種の核)/月支=「それがどう顕れるか」(タイミング・出方・温度)。P3生成では②の行だけをプロンプトに渡し、①は品質検証用の根拠として保持
2. **蔵干の面白い含み**を体感語に反映済み: 寅が丙を蔵す(火種ごと進む)、辰が癸乙を蔵す(混ざりを束ねる)、戌が丁を蔵す(火種を守る)、亥が甲を蔵す(水中の種木=仕込み)——など、単なる五行キャラ付けでは出ない縦深を本体系のZOUKAN_DETAILから採った
3. **確定後の格納先**: `core/kojindo.py` に `GETSU_SHI_APPEARANCE`(凍結定数・出典コメント付き)として実装予定
