# 古神道v3 Phase 3 設計案: 意識の種120本バッチ生成プロンプト

- 提出: こどたん → くろたん事前レビュー(指令書P3「生成開始前に提出」準拠)
- 日付: 2026-07-04

---

## 1. 生成マトリクス

**母体10本(骨格版・確定)× 月支12 = 120本**

- 各セルの出力: 「意識の種フレーズ」(骨格版と同型: 一行の核句——参拝前の具体アクション1つ)
- custom_id: `{god_id}_{支}`(例: `ninigi_子`)——god_id凍結表準拠

## 2. 前提素材が1つ不足しています(先に確定が必要)

**月支12種の「顕れ方」定義**がまだ存在しません。母体(十干=何を意識するか)に対し、月支は「その意識がどう顕れるか」の補助レイヤーですが、12支それぞれの性質定義(例: 子=夜明け前の始動/静かな水、午=正午の火/直進…)を決めないと、120本の軸がブレます。

**→ こどたん草案(12支×2行の定義表)を本提案の付録として次レビューで提出します。くろたん確定後に生成着手。**(算命学の十二支の伝統的性質を母体にし、骨格版と同じく「体感語」に変換する方針)

## 3. プロンプト構造(骨子)

### system(全120リクエスト共通・キャッシュ効率最大化)

```
あなたは「占いモンスターくろたん」の古神道v3・意識の種ライター。

## 血統ルール(最重要)
- 母体の「意識の種」の核概念を必ず保持する。別の人格・別の教えを創作しない
- 母体のフレーズの言い換えではなく、月支の顕れ方で「具体化」する

## トーン(宗教勧誘の回避・指令書4章)
- 「〜すべき」「必ず」「ご利益」「救われる」等の断定・利益保証は禁止
- 命令形ではなく「〜してから鳥居をくぐる」等の提案形
- 宗教の勧誘ではなく、文化的な所作の案内として書く

## 形式
- フレーズ: 20字以内の一行(「」で括れる強度)
- アクション: 参拝前に1分でできる具体的所作を1つ(数値・五感を含む)
```

### user(セルごと・可変部)

```
守護神: {god_name}({十干}・{story_type})
母体の意識の種: {骨格版の意識の種・全文}
月支: {支}——{確定済みの顕れ方定義}
この月支生まれの人向けに、母体を保ったまま顕れ方を specialize した
意識の種を1本。JSON: {"phrase": "...", "action": "...", "note": "..."(任意)}
```

### 出力制御

- 構造化出力(`output_config.format` JSON Schema・strict)で破損ゼロを機械保証
- `max_tokens`: 1,000(思考込み・短文出力のため十分)
